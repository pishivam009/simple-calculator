1. Install Python by running the installer "python-3.7.3-amd64.exe"
2. Install Robot Framework using:
    pip install robotframework
3. Install RIDE using:
    pip install robotframework-ride

Alternative:
1. Install Python by running the installer "python-3.7.3-amd64.exe"
2. Copy the file "requirements.txt" to you workspace
3. Install all packages using:
    pip install -r requirements.txt

Finally, open cmd and run:
    ride.py
